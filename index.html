Instrucciones:

Para usar los filtros:

Carga una foto

Usa siguiente para ver los filtros

Para los filtros personalizados use estas variables:

varying vec2 vTextureCoord;

uniform sampler2D uSampler;

uniform vec2 u_resolution;

uniform vec2 u_mouse;

uniform float u_time;

Si cargas un shader propio y quieres pasar a filtros animados, dale a "Siguiente filtro" para que se cargue la animacion
